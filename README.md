# solid-lanyard

Primitive that makes it easy to implement the Lanyard API into your Solid app

## Quick start

Install it:

```bash
npm i solid-lanyard
# or
yarn add solid-lanyard
# or
pnpm add solid-lanyard
```

Use it:

```tsx
import solid-lanyard from 'solid-lanyard'
```
